Neural Networks

A neural network is a special type of algorithm which mimics the structure of the human brain. They are used for deep learning, a type of machine learning. Machine learning simply refers to algorithms that "learn" as they are exposed to more data, and is a form of AI. Machine learning is used in many fields for many purposes, for example, to detect COVID-19 from a recording of a cough!

Neural networks are made up of many artificial neurons, each of which have inputs and an output. These inputs are "weighted", which means that they are multiplied with number. This controls how much each input affects the neuron. By tweaking these weights, the network can "learn". This is much like how pathways in our brain become more sensitive the more they are used. At the center of a neuron is the summation function, which combines the inputs using complicated math to create the output. The output may also go through an activation function, a sort of threshold.

These neurons are organized in "layers". Each layer consists of some number of neurons, whose outputs are the inputs of the next layer. There are different types of layers which are suitable for different purposes. These layers are then arranged to form an neural network/model.

Convolutional Neural Networks

For my project, I created/trained a convolutional neural network. Convolutional neural networks (CNNs) are a specific type of neural network that uses convolution layers. These layers excel at image processing, and are used for reducing the data to a simpler form while recognizing and extracting key features. Essentially, a "kernel" scans through the image, and uses a summation formula to combine adjacent pixels. Each group of pixels it combines becomes one pixel in the output. It does this in 2D space, as opposed to as a 1D list.

In between the convolutional layers, there are "pooling layers", which also simplify the data but in a more direct way. At the end of the model, there are also usually a few normal layers to convert the data into a good output form.
